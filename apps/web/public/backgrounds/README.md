# Images de fond studio

Placez ici les images d'illustration pour les fonds studio.

Les noms de fichiers doivent correspondre aux IDs des fonds :

- `studio_white.jpg` - Blanc Studio
- `studio_gray.jpg` - Gris Studio  
- `gradient_soft.jpg` - Dégradé Doux
- `outdoor_street.jpg` - Rue
- `lifestyle_cafe.jpg` - Café

**Format recommandé :**
- Extension : `.jpg` ou `.webp`
- Dimensions : 144x96 pixels minimum (ratio 3:2)
- Taille : <50KB pour un chargement rapide

Les images seront affichées dans les cartes de sélection de fond sur la page d'accueil.
